This is a daily report of complaints in Reliance. This excel file containts the following sheets: 1. ACKs Mitig - This is a list of complaints where a decision has been made and no MedWatch has been submitted or a MedWatch has been submitted, but the third_acknowledgement failed. 2. Additionally it checks if the medwatch exists OR if the medwatch has not been submitted or not completed. And it also makes sure the complaint is not completed or voided. 3. Potential Supps-Upgrades - This is a list of complaints that require a supplemental or the complaint has been upgraded, requiring a submission of the initial MedWatch.

Long Description HTML:

This is a daily report of complaints in Reliance. This excel file containts the following sheets:

  • ACKs Mitig:
    • This is a list of complaints where a decision has been made and no MedWatch has been submitted or a MedWatch has been submitted, but the third_acknowledgement failed.
  • Not Complete:
    • This is a list of reportable and potentially reportable complaints where a decision has not been made yet OR medical intervention or death is marked as Yes. Additionally it checks if the medwatch exists OR if the medwatch has not been submitted or not completed. And it also makes sure the complaint is not completed or voided.
  • Potential Supps-Upgrades:
    • This is a list of complaints that require a supplemental or the complaint has been upgraded, requiring a submission of the initial MedWatch

Data: Reliance Only
